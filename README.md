# Hafzan FRM Guide — Part I

**The 20% of FRM Part I that covers 80% of the exam.**

A free, focused, single-file study tool for FRM Part I candidates. No signup. No install. Just open the file and study.

---

## What it is

FRM Part I covers hundreds of pages of material across four sections. This app cuts through the noise — isolating the eight highest-yield topics, presenting every key formula with clear annotations, and testing you with exam-style questions and full worked solutions.

Built on the 80/20 principle: master these topics cold and you walk into the exam covering the majority of what's tested.

## How to use

**Option 1 — Local (simplest):**
1. Download `index.html`
2. Double-click to open in your browser
3. Done

**Option 2 — GitHub Pages:**
1. Fork this repo
2. Go to Settings → Pages → Source → main branch
3. Your copy goes live at `https://yourusername.github.io/hafzan-frm`

No Node. No npm. No build step. One file.

---

## How it works

Each topic contains four sections:

- **Concept** — the intuition behind the topic
- **Formula** — every key formula with variable annotations
- **Practical Use** — real-world application and common exam traps
- **Practice Question** — click an answer to reveal a full worked solution

Navigate using the sidebar or the Previous / Next buttons.
